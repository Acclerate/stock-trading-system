# 市场监控技能

实时监控大盘、板块、外围市场的整体情况，给出市场环境评分和操作建议。

## 使用方式

```
/market-monitor                          # 完整市场监控
/market-monitor --overseas               # 仅外围市场
/market-monitor --indices                # 仅A股指数
/market-monitor --sectors                # 仅板块监控
/market-monitor --sentiment              # 仅市场情绪
```

## 执行流程

### 1. 外围市场扫描

获取美股、欧股、港股、期货等外围市场数据：

```bash
!python -c "
import requests
import json

# 这里应该调用实际的数据API
# 暂时返回模拟数据
data = {
    'dow': '+0.5%',
    'nasdaq': '+0.8%',
    'sp500': '+0.6%',
    'europe': '+0.3%',
    'a50': '+0.2%',
    'comment': '外围市场普涨，利好A股'
}
print(json.dumps(data, ensure_ascii=False))
"
```

**评分标准 (0-10分)**：
- 基础分：5分
- 利好/普涨：+2分
- 利空/普跌：-2分

### 2. A股大盘分析

调用 Python 脚本获取指数数据：

```bash
!cd "D:/privategit/github/stockScience" && python data_sources.py --market
```

**分析要点**：
- 三大指数点位和涨跌幅
- 成交量变化
- 涨跌家数比
- 北向资金流向

**评分标准 (0-10分)**：
- 基础分：5分
- 涨跌家数比 > 0.6：+2分
- 成交量 > 10000亿：+1分
- 涨跌家数比 < 0.4：-2分

### 3. 板块监控

关注重点板块：
- **风电板块**：金风科技所在板块
- **商业航天**：蓝箭航天相关概念
- **新能源**：整体赛道

**评分标准 (0-10分)**：
- 基础分：5分
- 关注板块≥2个上涨：+2分
- 关注板块≥1个上涨：+1分

### 4. 市场情绪

**分析指标**：
- 恐贪指数
- 换手率
- VIX指数

**评分标准 (0-10分)**：
- 基础分：5分
- 恐贪指数 > 60：+2分
- 恐贪指数 < 40：-2分
- VIX < 20：+1分

## 注意事项

1. 数据时效性：网络数据可能有5-15分钟延迟
2. 综合判断：不能只看单一指标，要综合分析
3. 动态调整：市场变化快，需要持续监控
4. 风险提示：即使评分高也要注意风险控制
