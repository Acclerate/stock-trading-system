# 每日复盘技能

每日、每周、每月交易复盘，总结经验教训，优化交易系统。

## 使用方式

```
/daily-review                           # 每日复盘
/daily-review --weekly                  # 每周复盘
/daily-review --monthly                 # 每月复盘
/daily-review --date 2026-02-07         # 指定日期复盘
```

## 复盘核心原则

1. **客观真实**：不隐瞒，不美化
2. **深度分析**：找原因，不找借口
3. **总结规律**：发现模式，提炼经验
4. **持续改进**：每周优化系统
5. **长期主义**：不追求短期暴利

```
□ 今天遵守交易纪律了吗？

□ 今天执行止损/止盈了吗？
```
