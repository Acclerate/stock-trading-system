# 个股分析技能

对个股进行深度技术分析和基本面分析，给出综合评分和操作建议。

## 使用方式

```
/stock-analyzer 300476                    # 分析胜宏科技
/stock-analyzer 002202                    # 分析金风科技
/stock-analyzer --list                    # 分析自选股列表
```

## 执行流程

### 1. 获取股票基础数据

调用 Python 脚本获取股票数据：


### 2. 技术分析 (40% 权重，70分)

分析以下指标并评分：

| 指标 | 评分标准 | 分值 |
|------|----------|------|
| 趋势 | 多头排列/震荡/空头排列 | 0-10 |
| MACD | 金叉/死叉/零轴位置 | 0-10 |
| RSI | 超买/超卖/强势/弱势 | 0-10 |
| KDJ | 金叉/死叉/超买/超卖 | 0-10 |
| 布林带 | 突破上轨/下轨/通道内 | 0-10 |
| 量能 | 放量/缩量/平稳 | 0-10 |
| 形态 | 突破/反转/整理/下跌 | 0-10 |

**技术面总分**：___/70 → 加权后 ___/40

### 3. 基本面分析 (20% 权重，20分)

- 公司基本面：业务、行业地位、竞争优势
- 财务数据：营收增长、利润增长、ROE、负债率
- 估值水平：PE、PB 行业分位
- 业绩预期：机构预测、业绩预告

**基本面评分**：___/20

### 4. 资金面分析 (20% 权重，20分)

- 主力资金：净流入/流出、大单买卖比
- 散户资金：情绪、跟风程度
- 北向资金：持仓变化、买卖情况
- 机构动向：评级调整、研报观点

**资金面评分**：___/20

### 5. 市场环境 (20% 权重，20分)

- 大盘：走势、强度
- 板块：活跃度、轮动
- 情绪：恐贪指数、市场情绪

**环境评分**：___/20

## 操作建议映射

| 综合评分 | 操作建议 | 目标价位 |
|----------|----------|----------|
| 80-100 | 强烈买入 | 买入区间：现价±2%<br>止损：-10%<br>止盈：+20% |
| 60-80 | 买入 | 买入区间：现价±2%<br>止损：-10%<br>止盈：+20% |
| 40-60 | 观望 | 等待更好的入场点 |
| 20-40 | 谨慎 | 考虑减仓或观望 |
| 0-20 | 回避 | 不建议操作 |

## 分析原则

1. **客观性**：基于数据和指标，不凭感觉
2. **全面性**：技术+基本面+资金面综合分析
3. **及时性**：使用最新数据
4. **风险意识**：永远提示风险
5. **逻辑性**：给出明确的逻辑链条
