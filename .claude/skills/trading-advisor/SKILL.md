# 交易顾问技能

提供具体的交易操作建议，包括买入、卖出、观望的决策。

## 使用方式

```
/trading-advisor buy 002202                   # 买入咨询
/trading-advisor sell 002202                  # 卖出咨询
/trading-advisor buy 002202 --price 25.5     # 指定价格
```

## 决策流程

### 第一步：收集用户信息

必填信息：
- 股票代码：$ARGUMENTS[1]
- 操作类型：$ARGUMENTS[0] (buy/sell)
- 当前价格：询问或从参数获取
- 可用资金：询问或从参数获取
- 当前持仓：如为卖出操作

### 第二步：多维评估

```
┌─────────────────────────────────────────┐
│ 【交易资格评估】                         │
├─────────────────────────────────────────┤
│                                         │
│  1. 技术面评估（40%）                    │
│     └─ 评分≥30分？                      │
│                                         │
│  2. 风险评估（30%）                     │
│     └─ 风险可控？                       │
│                                         │
│  3. 市场环境评估（20%）                 │
│     └─ 环境适合？                       │
│                                         │
│  4. 基本面评估（10%）                   │
│     └─ 逻辑合理？                       │
│                                         │
└─────────────────────────────────────────┘

【任一维度不达标】
→ 禁止交易或降低仓位
```

### 第三步：买入决策树

```
满足所有买入条件？
    ├─ NO ──→ 禁止买入
    │
    └─ YES
      ├─ 风险收益比≥3:1？
      │   ├─ NO ──→ 降低仓位或放弃
      │   └─ YES
      │     ├─ 仓位≤20%？
      │     │   ├─ NO ──→ 限制仓位到20%
      │     │   └─ YES
      │     │     ├─ 是否追高？
      │     │     │   ├─ YES ─→ 禁止买入
      │     │     │   └─ NO
      │     │     │     └─→ 【批准买入】
```

### 第四步：卖出决策树

```
触发卖出条件？
    ├─ 止损触发？
    │   ├─ YES ──→ 立即卖出
    │   └─ NO
    │
    ├─ 止盈触发？
    │   ├─ YES ──→ 分批卖出
    │   └─ NO
    │
    ├─ 技术面转坏？
    │   ├─ YES ──→ 立即卖出
    │   └─ NO
    │
    └─ 未触发 ──→ 继续持有
```

## 买入条件清单

```
技术条件（必须满足≥3项）：
  □ MA多头排列
  □ MACD金叉
  □ RSI在50以下或刚超卖回升
  □ 放量突破压力位
  □ 回调到支撑位企稳
  □ 布林带下轨附近
  □ K线突破形态

风险条件（必须满足）：
  □ 风险收益比≥3:1
  □ 最大亏损≤10%
  □ 单笔仓位≤20%
  □ 总仓位≤80%

市场条件（必须满足）：
  □ 大盘非单边大跌
  □ 板块无重大利空
  □ 市场情绪稳定或偏乐观

禁止条件（有任一即禁止）：
  ✗ 连续涨停后
  ✗ 已经大涨（短期>30%）
  ✗ 技术面明显走坏
  ✗ 重大利空消息
  ✗ 公司基本面恶化
```

## 卖出条件清单

```
强制卖出（触发任一立即执行）：
  ☢ 跌破止损价（-10%）
  ☢ MACD死叉且放量下跌
  ☢ 跌破关键支撑位
  ☢ 放量暴跌
  ☢ 重大利空消息
  ☢ 基本面恶化

部分卖出（触发考虑执行）：
  □ 达到止盈价（+20%）
  □ 远离均线>10%
  □ 放量滞涨
  □ 连续大涨后
  □ 持仓>3个月未达目标

继续持有（未触发卖出条件）：
  □ 技术面良好
  □ 趋势向上
  □ 未达目标价位
  □ 无重大利空
```

## 仓位管理规则

```
┌─────────────────────────────────────────┐
│ 【仓位管理矩阵】                         │
├─────────────────────────────────────────┤
│                                         │
│  技术强 + 基本面强 → 30%               │
│  技术强 + 基本面中 → 20%               │
│  技术强 + 基本面弱 → 10%               │
│  技术中 + 基本面强 → 20%               │
│  技术中 + 基本面中 → 10%               │
│  技术中 + 基本面弱 → 观望              │
│  技术弱 + 基本面强 → 10%               │
│  技术弱 + 基本面中 → 观望              │
│  技术弱 + 基本面弱 → 禁止              │
│                                         │
└─────────────────────────────────────────┘

特殊情况：
  - 首次建仓：≤20%
  - 加仓操作：≤10%
  - 总仓位上限：80%
  - 现金保留：≥20%
```

## 核心原则

1. **安全第一**：宁可错过，不可做错
2. **纪律优先**：计划一旦制定，必须执行
3. **风险可控**：每笔交易必须设止损
4. **仓位控制**：永远不要满仓一只股票
5. **客观理性**：基于数据和逻辑，不凭情绪
